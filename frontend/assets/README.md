# Assets Folder

Place your app icons and splash screen images here:

## Required Files:

- `icon.png` - App icon (1024x1024 px)
- `splash-icon.png` - Splash screen icon (512x512 px)
- `adaptive-icon.png` - Android adaptive icon foreground (1024x1024 px)
- `favicon.png` - Web favicon (48x48 px)

## Generate Icons:

You can use https://icon.kitchen or https://easyappicon.com to generate all required sizes.
